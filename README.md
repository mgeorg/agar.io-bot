# agar.io-bot
An agar.io bot which uses a chrome extension to ineject modified javascript files which must be hosted on a remote server.

# running the server:
first time setup
`cd server`
`virtualenv venv`
`source venv/bin/activate`
`pip install -r app/requirements.txt`
start the server
`source venv/bin/activate`   
`python app/app.py`
